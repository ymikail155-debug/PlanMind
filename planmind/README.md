PlanMind - Flutter proje (MVP reklamlı)

İçerikler:
- Türkçe & İngilizce destek
- Meslek seçimi, modül seçimi
- Banner reklam (AdMob test ID)
- Rewarded video (AdMob test ID) placeholder
- Hive ile basit offline kayıt

Kullanım:
1) Flutter SDK yüklü olduğundan emin olun.
2) Proje klasöründe terminal açın:
   flutter pub get
3) Emulator veya gerçek cihaz bağlayın:
   flutter run

AdMob:
- Test modunda banner ve rewarded ad kullanılmıştır.
- Yayına alırken google_mobile_ads içindeki testAdUnitId'leri kendi AdMob ID'leriniz ile değiştirin.
